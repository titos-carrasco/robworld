Para compilar el archivo java, ingresar el siguiente comando en una consola:

    $ javac -cp "../../dist/robworld-1.1.5.jar:../../dist/javax.json-1.1.4.jar:."  *.java
    $ java -cp "../../dist/robworld-1.1.5.jar:../../dist/javax.json-1.1.4.jar:." archivo_class


*** En windows reemplazar el caracter ':' por ';'
