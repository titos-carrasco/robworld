Para compilar el archivo java, ingresar el siguiente comando en una consola:

    $ javac -cp "../../dist/robworld-x.y.z.jar:../../dist/javax.json-1.1.4.jar:."  *.java
    $ java -cp "../../dist/robworld-x.y.z.jar:../../dist/javax.json-1.1.4.jar:." archivo_class


*** En windows reemplazar el caracter ':' por ';'
